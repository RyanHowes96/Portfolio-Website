SafePoint Mobile App UX Project

UX Development of a Mobile Navigation App designed around safety.

An intuitive mobile application designed around finding the safest routes between two different points. Making journeys, especially at night safer and more comfortable for vulnerable people.

Once a location is selected, the application will create a route between the user and the destination highlighting the safest route. This will be done by analysing crime statistics in areas, as well as utilising traffic data to determine busy and well lit roads. The application will avoid taking the user down dark and secluded areas, which again will be determined by traffic data, as the roads and paths used less tend to have higher risk of an incident occurring.
